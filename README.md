# GitHub Pages 个人主页

这是一个可以直接发布到 GitHub Pages 的静态学术主页，包含中英文简历下载链接。

## 发布方法

1. 注册或登录 [GitHub](https://github.com/)。
2. 新建一个**公开（Public）**仓库，仓库名必须是 `你的GitHub用户名.github.io`。
3. 把以下三个文件上传到仓库根目录：`index.html`、`cv-en.pdf`、`cv-zh.pdf`。
4. 进入仓库的 **Settings → Pages**，在 **Build and deployment** 中选择 **Deploy from a branch**，分支选 `main`，目录选 `/ (root)`，点击保存。
5. 等待几分钟后打开 `https://你的GitHub用户名.github.io/`。

发布后，这个 HTTPS 链接可以作为 OpenReview 的个人主页链接。

## 发布前需要修改

当前版本已经填入郑乐言 / Leyan Zheng、浙江大学经济学院 PhD Candidate、研究方向和邮箱。主页只保留个人介绍、研究兴趣和中英文简历下载链接。

为适合公开网页，简历 PDF 默认没有放手机号、详细住址、政治面貌、民族、出生年月和照片；如需保留这些信息，可以在公开上传前自行决定是否补回。

GitHub Pages 的站点默认公开，主页中不要放身份证号、家庭住址、私人电话等敏感信息。
